Same 5 screens, 16:9 ratio

## How to Capture
```
npx expo start --ios           # open in iOS Simulator
# CMD+S to take screenshot → saved to Desktop
npx expo start --android       # open in Android Emulator
# Use emulator toolbar → camera icon → screenshot
```

## Screenshot Tips
- Use dark mode (already default in app)
- Show mock data loaded (the app shows mock data by default — no API needed)
- Capture with status bar time set to 9:41 AM (iOS: xcrun simctl status_bar)
- Android: use a Pixel 8 emulator for Play Store requirements
